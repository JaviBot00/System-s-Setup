# ProTips - Setup Ubuntu

This repository contains scripts and instructions for setting up a fresh Ubuntu installation with recommended packages and configurations.

## Contents

- `setupUbuntu/main.sh`: Main script to choose between desktop and server setup.
- `setupUbuntu/setup-desktop.sh`: Script for setting up a desktop environment with essential packages and configurations.
- `setupUbuntu/setup-dev.sh`: Script for setting up a development environment with programming tools and libraries.
- `setupUbuntu/setup-graphics.sh`: Script for setting up a desktop environment with graphics-related packages.
- `setupUbuntu/setup-server.sh`: Script for setting up a server environment with essential server packages.

## Usage

1. Clone the repository:

   ```bash
   https://github.com/JaviBot00/System-s-Setup.git
    cd System-s-Setup/setupUbuntu
    ```

2. Make the main script executable:

    ```bash
    chmod +x main.sh
    ```

3. Run the main script:

    ```bash
    ./main.sh
    ```

4. Follow the on-screen prompts to select the desired setup options.

## Requirements

- A fresh installation of Ubuntu 24.04 or later.
- Internet connection for downloading packages.

## Disclaimer

Use these scripts at your own risk. Always review the scripts before executing them to ensure they meet your specific needs and security requirements.
